Brain Trainer

Brain Trainer es una aplicación Android diseñada para mejorar habilidades cognitivas como memoria, atención y velocidad de procesamiento mediante minijuegos interactivos.

El proyecto destaca por ofrecer una experiencia accesible, sin bloquear funcionalidades esenciales detrás de pagos, apostando por un modelo no intrusivo.

Tecnologías utilizadas

  • Kotlin
  • Jetpack Compose
  • Firebase (Authentication, Firestore DB)
  • Credential Manager
  • Hilt

Características principales

  • Minijuegos interactivos
  • Seguimiento del progreso del usuario
  • Persistencia de datos
  • Navegación entre pantallas optimizada

Arquitectura y buenas prácticas

  • Clean Architecture
  • Principios SOLID
  • Modelo-Vista-Controlador (MVC)

¿Qué demuestra este proyecto?

  • Desarrollo completo de una app desde cero
  • Integración de autenticación y base de datos
  • Aplicación de buenas prácticas y arquitectura limpia
  • Uso de control de versiones con Git
